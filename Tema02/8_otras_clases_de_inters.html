8. Otras clases de interés

Clase Math

https://docs.oracle.com/javase/8/docs/api/java/lang/Math.html

La clase Math reside en el paquete java.lang y proporciona un conjunto de métodos estáticos para realizar varios cálculos matemáticos diferentes. Al ser una clase estática no se pueden instanciar objetos, así que para hacer uso de sus métodos se pone el nombre de la clase seguido de un punto y el  nombre del método.

Math.metodo();

Atributos:

  • static double E  -->  representa el número irracional   e (2.718281828459045).
  • static double PI  -->  representa el valor pi (3.141592653589793).

Métodos:

  • Trigonometría:
    • static double acos(double a)  -->  arcoseno de un ángulo.
    • static double asin(double a)  -->  arcoseno de un ángulo.
    • static double atan(double a)  -->  arcotangente
    • static double atan2(double y, double x)  -->  Devuelve el ángulo theta de la conversión de coordenadas rectangulares (x, y) a coordenadas polares (r, theta).
    • static double cos(double a)  -->  coseno de un ángulo
    • static double cosh(double x)  -->  coseno hiperbólico
    • static double sin(double a)  -->  devuelve el seno de un ángulo
    • static double sinh(double x)  -->  devuelve el seno hiperbólico
    • static double tan(double a)  -->  tangente de un ángulo.
    • static double tanh(double x)  -->  tangente hiperbólica.
    • static double toDegrees(double angrad)  -->  convierte un angulo expresado en radianes a grados.
    • static double toRadians(double angdeg)  -->  convierte un ángulo de grados a radianes.
  • Redondeo, truncamiento:
    • static double ceil(double a)  -->  Redondea hacia arriba, es decir, devuelve el entero más pequeño que es mayor o igual al número dado.
    • static double floor(double a)  -->  Redondea hacia abajo, es decir, devuelve el entero más grande que es menor o igual al número dado.
    • static int floorDiv(int x, int y)  -->  Devuelve el valor int más grande (más cercano al infinito positivo) que sea menor o igual al cociente algebraico.
      static long floorDiv(long x, long y)
    • static int floorMod(int x, int y)  -->  Devuelve el módulo  de los argumentos int ( x % y )
    • static long floorMod(long x, long y)
    • static double nextAfter(double start, double direction)  -->  Devuelve el número adyacente a start en la dirección de direction.
    • static float nextAfter(float start, double direction)
    • static double nextDown(double d)  -->  devuelve el número inferior.
    • static float nextDown(float f)
    • static double nextUp(double d)  -->  devuelve el número superior.
    • static float nextUp(float f)
    • static double rint(double a)  -->  Devuelve el valor double más cercano en valor al argumento y igual a un entero matemático.
    • static long round(double a)  -->  redondea al entero más próximo.
    • static int round(float a)
  • Potencias, raices, logaritmos:
    • static double cbrt(double a)  --> raíz cúbica.
    • static double exp(double a)  -->   Devuelve el número de Euler, e, elevado a la potencia de un valor doble.
    • static double expm1(double x)  -->  devuelve ex-1
    • static int getExponent(double d)  -->  Devuelve el exponente utilizado en la representación de un doble
    • static int getExponent(float f)
    • static double log(double a)  -->  devuelve el logaritmo en base e.
    • static double log10(double a)  --> devuelve el logaritmo en base 10.
    • static double pow(double a, double b)  -->  devuelve ab.
    • static double sqrt(double a)  -->  devuelve la raiz cuadrada positiva.
  • máximos, mínimos, siguientes, anteriores, sumas, restas, multiplicaciones, divisiones:
    • static int addExact(int x, int y)  -->  suma de dos argumentos.
    • static long addExact(long x, long y)
    • static int decrementExact(int a)  -->  Devuelve el argumento disminuido en uno, lanzando una excepción si el resultado desborda un int.
    • static long decrementExact(long a)
    • static int incrementExact(int a)  -->  devuelve el argumento incrementado en 1 (produce excepción si hay desbordamiento).
    • static long incrementExact(long a)
    • static double max(double a, double b)  -->  mayor de 2 valores.
    • static float max(float a, float b)
    • static int max(int a, int b)
    • static long max(long a, long b)
    • static double min( a,  b)  -->  menor de 2 valores
    • static int multiplyExact( x,  y)  -->  devuelve la multiplicación. Si soprepasa el máximo valor produce una excepción.
    • static int subtractExact(int x, int y)  -->  diverencia x - y. Si se produce overflow produce una excepción.
    • static long subtractExact(long x, long y)
  • Otras:
    • static double abs(double a)  -->  valor absoluto
    • static float abs(float a)
    • static int abs(int a)
    • static long abs(long a)
    • static double copySign(double magnitude, double sign)  -->  Devuelve el primer argumento de punto flotante con el signo del segundo argumento de punto flotante.
    • static float copySign(float magnitude, float sign)
    • static double random()  -->  devuelve un número aleatorio entre [0.0 y 1.0[
    • static double signum(double d)  -->  devuelve el signo: 0 si d es cero; 1 si d es positivo; -1 si d es negativo.
    • static float signum(float f)
    • static int toIntExact(long value)  -->  convierte a int. Si hay overflows produce una excepción.

Ejemplo:

public class App {
  public static void main(String[] args) {
    double valor = 2.3265874521365;
    int base = 5, exponente = 3;
    System.out.println("Número: " + valor);
    System.out.println("Redondea a 2 decimales hacia abajo: " + Math.floor(valor * 100)/100.0);
    System.out.println("Redondea a 2 decimales hacia arriba: " + Math.ceil(valor * 100)/100.0);
    System.out.println("Redondea a 2 decimales: " + Math.round(valor * 100)/100.0);
    System.out.println(base + " elevado a " + exponente + " = " + Math.pow(base, exponente));
    System.out.println("Tirada de dado: " + (int)(1 + Math.round((Math.random() * (6 - 1)))));
  }
}

Clase Random

https://docs.oracle.com/javase/8/docs/api/java/util/Random.html

Se utiliza para generar un flujo de números pseudoaleatorios. La clase utiliza una semilla de 48 bits, para generar los números. Si se crean dos instancias de Random con la misma semilla y se realiza la misma secuencia de llamadas de método para cada una, generarán y devolverán secuencias idénticas de números. Se encuentra en la librería java.util

Constructores:

  • Random()  -->  crea un objeto Random
  • Random(long seed)  -->  crea un objeto utilizandouna semilla.

Métodos:

  • DoubleStream doubles()  -->  devuelve un stream ilimitado de valores double pseudoaleatorios entre 0 (inclusive) y 1 (excludo), [0, 1[
  • DoubleStream doubles(double origen, double limite)  -->  devuelve un stream ilimitado de valores entre [origen, limite[
  • DoubleStream doubles(long streamSize)  -->  devuelve un stream de doubles con un número concretode valores entre [0, 1[
  • DoubleStream doubles(long tamaño, double origen, double limite)
  • IntStream ints() -->  devuelve un stream ilimitado de enteros.
  • IntStream ints(int origen, int limite) -->  stream de enteros entre [origen, limite[
  • IntStream ints(long n)  --> stream de enteros con  n  enteros.
  • IntStream ints(long n, int origen, int limite)
  • LongStream longs()
  • LongStream longs(long streamSize)
  • LongStream longs(long randomNumberOrigin, long randomNumberBound)
  • LongStream longs(long streamSize, long randomNumberOrigin, long randomNumberBound)
  • protected int next(int bits)  --> genera el siguiente número pseudoaleatorio.
  • boolean nextBoolean()  -->  devuelve el siguiente boolean de forma pseudoaleatoria.
  • void nextBytes(byte[] bytes)  -->  Genera bytes aleatorios y los coloca en una matriz de bytes proporcionada por el usuario.
  • double nextDouble()  -->  devuelve el siguiente double ( entre [0, 1[ ) de forma pseudoaleatoria.
  • float nextFloat()
  • int nextInt()
  • int nextInt(int bound)  --> siguiente entero entre [0, bound[
  • long nextLong()
  • void setSeed(long seed)  -->  Establece la semilla de este generador de números aleatorios utilizando una semilla pasada como parámetro

Ejemplo:

import java.util.Random;
public class App {
  public static void main(String[] args) {
    Random randomNumbers = new Random();       // create random object
    System.out.println("---------- Boolean Aleatorio ---------" );
    /* Devuelve el siguiente boolean pseudoaleatorio que se usa en un sorteo  */
    boolean valor = randomNumbers.nextBoolean();
    System.out.println("Valor del booleano aleatorio: " + valor);
    /* Simula la tirada de un dato, dando valores entre 1 y 6. Dado que genera números entre [0 y 6[
    * al sumarle 1, el más pequeño será 1. Y puesto que 6 no sale nunca (el int más pequeño es 5), 5 + 1 dará 6 */
    System.out.println("\n---------- Tirada de 5 dados ---------" );
    System.out.println("Dado 1: " + (randomNumbers.nextInt(6)+1) );
    System.out.println("Dado 2: " + (randomNumbers.nextInt(6)+1) );
    System.out.println("Dado 3: " + (randomNumbers.nextInt(6)+1) );
    System.out.println("Dado 4: " + (randomNumbers.nextInt(6)+1) );
    System.out.println("Dado 5: " + (randomNumbers.nextInt(6)+1) );

    // devuelve el siguiente pseudoaleatorio long
    Long val = randomNumbers.nextLong();
    System.out.println("\n---------- Long aleatorio ---------" );
    System.out.println("valor Long aleatorio: " + val);      
  }
}

Clase DecimalFormat

https://docs.oracle.com/javase/8/docs/api/java/text/DecimalFormat.html

DecimalFormat ( forma parte del paqueta java.text  ) es la clase java nos permite mostrar los números en pantalla con el formato que queramos, por ejemplo, con dos decimales, con una coma para separar los decimales, etc. DecimalFormat también es útil para presentar un número en un JTextField o recoger el texto del JTextField y reconstruir el número.

Para dar formato a un número, tenemos que definir un patrón, que es una secuencia de caracteres especiales que pueden mezclarse con texto. Los caracteres especiales son:

0   –->   escribe el dígito suministrado, 0 si no se suministra ninguno.
#   –->   escribe el dígito suministrado, nada si no se suministra dígito.
.   –->   indica dónde poner el separador de decimales.
,   –->   indica dónde poner el separador de miles.
-    -->   signo menos
E  -->   Separa la mantisa y el exponente en notación científica. No es necesario ponerlos entre comillas en posición prefijo o sufijo.
;  -->   separa patrones para números positivos y negativos.
%  -->  lo multiplica por 100 y lo muestra como porcentaje.
\u2030  -->  Multiplicar por 1000 y mostrar como valor por mil
¤  -->  símbolo de moneda

Cuando el patrón se aplica a un número, se ejecutan sus reglas de formato y el resultado se imprime de acuerdo con el DecimalFormatSymbol de la configuración regional de nuestra JVM, a menos que se especifique una configuración regional específica.

Constructores:

  • DecimalFormat()  -->  crea un objeto usando el patrón y símbolos del formato local.
  • DecimalFormat(String pattern)  -->  crea un objeto usando el patrón pasado y los símbolos locales.
  • DecimalFormat(String pattern, DecimalFormatSymbols symbols)  -->  crea un objeto usando el patrón y símbolos pasados.

Métodos:

  • void applyLocalizedPattern(String pattern)  -->  aplica el patrón.
  • void applyPattern(String pattern)  -->  aplica el patrón.
  • StringBuffer format(double number, StringBuffer result, FieldPosition fieldPosition)  -->  formatea un double
  • StringBuffer format(long number, StringBuffer result, FieldPosition fieldPosition)  -->  formatea un long
  • StringBuffer format(Object number, StringBuffer toAppendTo, FieldPosition pos)  -->  formatea un objeto numérico 
  • Currency getCurrency()  -->  Obtiene la moneda utilizada por este formato decimal al formatear valores de moneda.
  • int getGroupingSize()  -->  muestra el número que se utiliza para agrupar los dígitos (en 1,123.658.235.23 están agrupados cada 3 dígitos).
  • int getMultiplier()  -->  devuelve el multiplicador usado en los porcentajes, miles y formatos similares.
  • String getNegativePrefix()  -->  obtiene el prefijo para valores negativos ( signo - )
  • String getPositivePrefix()  -->    obtiene el prefijo para valores positivos
  • void setGroupingSize(int newValue)  -->  establece el número de dígitos que se agruparán.
  • void setMultiplier(int newValue)  -->  establece el multiplicador usado en los porcentajes, miles, y formatos similares
  • void setNegativePrefix(String newValue)  -->  cambia el símbolo para los números negativos.
  • void setPositivePrefix(String newValue)  -->  cambia el símbolo para los números positivos.

Redondear decimales

DecimalFormat formateador = new DecimalFormat("####.##");
System.out.println(formateador.format(3.43242383));      // 3,43 
System.out.println(formateador.format(58754.43522383));  // 5875,.44 
System.out.println(formateador.format(1284.43492383));   // 1284,43

Las # representan una cifra, así que hemos puesto que el número se ponga en pantalla con cuatro cifras (si la parte entera tiene más de 4 igual se muestran), un punto decimal y dos decimales (se redondea solo en función del decimal inferior, no de un redondeo matemático).

Si usamos 0 en vez de #, los huecos se rellenarán con ceros por delante.

DecimalFormat formateador = new DecimalFormat("0000.00");
System.out.println(formateador.format(3.43242383));     // 3,43 
System.out.println(formateador.format(58754.43522383)); // 5875,44 
System.out.println(formateador.format(1284.43492383));  // 1284,43
System.out.println(formateador.format(1284.4));         // 1284,40

las cifras que falten por delante o por detrás, se rellenarán con ceros.

Mostrar porcentajes

DecimalFormat formateador = new DecimalFormat("00.##%");
System.out.println (formateador.format (0.43242383)); // 43,24% 
System.out.println (formateador.format (0.43522383)); // 43,52% 
System.out.println (formateador.format (1.43492383)); // 143,49%
System.out.println (formateador.format (1.4));        // 140%;

Notación científica

DecimalFormat formateador = new DecimalFormat("00.#######E0"); 
System.out.println (formateador.format (0432423.83));     // 43,242383E4 
System.out.println (formateador.format (-04352238.3));    // -43,522383E5 
System.out.println (formateador.format (0.000143492383)); // 14,3492383E-5

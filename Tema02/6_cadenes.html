6. Cadenes

Una cadena és una seqüència de caràcters. Java té una classe incorporada en el paquet java.lang que encapsula les estructures de dades d'una cadena. Aquesta classe, anomenada String, és la representació com a objecte d'una matriu de caràcters que no es pot canviar.

Podem crear un objecte de tipus cadena de dues formes diferents:

  • Com una variable de tipus primitiu.
String nom = "Alicia Giménez Bartlett";
  • Com  un objecte:
String nom = new String("Alicia Giménez Bartlett");

Encara que els dos mètodes són vàlids es recomana el primer ja que és més eficient.

Els literals de cadena van entre cometes: "Exemple de cadena", i poden utilitzar-se en qualsevol lloc que s'utilitze un objecte de tipus String.

Ús de cadenes

  • Concatenació: java permet concatenar cadenes fàcilment utilitzant l'operador  +.
String nom = "Juan";
String cognoms = "Lora Gisbert";
String nomComplet = cognoms + ", " + nom;
  • Métodes de cadenes: la classe String té una sèrie de mètodes per a treballar amb cadenes. Alguns d'ús freqüent són:

    • .length(): longitud de la cadena. Retorna el nombre de caràcters d'una cadena.
         String s = "abc";
         System.out.println(s.length());    // mostrarà 3

    • .charAt(índex): extracció de caràcters. Extrau un únic caràcter d'una cadena corresponent amb el caràcter la posició del qual ve indicada per índex. Has de tindre en compte que la primera posició és el 0.
         "abc".charAt(1);    // retornarà 'b'

    • .getChars(inici, fi, destí, posdest): extrau diversos caràcters de la cadena i els afig en destí. Cal especificar l'índex del primer caràcter i de l'últim més un que es desitgen copiar, a més de la matriu char on es desitgen col·locar aquests caràcters.
         String s = "Aixó no es una cançó";
         char buf[] = new char[2];
         s.getChars(5, 7, buf, 0);

    • .equals(cadena): compara amb la cadena passada com a paràmetre. Retorna true si les cadenes són iguals. En cas contrari retorna false.

    • .equalsIgnoreCase (cadena): igual que l'anterior però sense tindre en compte les majúscules i minúscules.
         String cadena1="Carmen";
         String cadena2="carmen";
         System.out.println(cadena1.equals(cadena2));              // false
         System.out.println(cadena1.equalsIgnoreCase(cadena2));    // true

      El mètode equals i l'operador == fan dues proves completament diferents per a la igualtat. Mentre que el mètode equals compara els caràcters continguts en una String, l'operador == compara dues referències d'objecte per a veure si es refereixen a la mateixa instància. Per tant, no podem usar el signe == perquè aquesta seria una comparació binària de punters a memòria i no ens retornaria el valor correcte.

    • compareTo( pcadena ): compara dues cadenes lexicogràficament i retorna un valor sencer. El valor retornat és 0 si totes dues cadenes tenen el mateix contingut; és un valor negatiu si pcadena és major; és un valor positiu si pcadena és menor.
         String apellido1 = "Monfort";
         String apellido2 = "Rubert";
         System.out.println(apellido1.compareTo(apellido2));   // mostra   -5

    • .toLowerCase(): converteix la cadena a minúscules.
    • .toUpperCase(): converteix la cadena a majúscules.

    • .toString(): converteix un objecte a una cadena o String. Totes les classes hereten toString() des de la classe Object i moltes classes del paquet java.lang sobreescriuen aquest mètode per a proporcionar una implementació més d'acord amb la pròpia classe. Per exemple, les classes Character, Integer, Boolean, etc.. sobreescriuen toString() per a proporcionar una representació en String dels objectes.

    • .valueOf(): permet convertir variables de diferents tipus a un String.
         System.out.println(String.valueOf(Math.PI));

    • substring(inici, final): retorna part d'una cadena. La cadena retornada començarà en la posició inici i acabarà en la posició final. Si final no s'indica serà l'últim caràcter de la cadena.
         String str="El lenguaje Java";
         String subStr=str.substring(12);    // subStr --> "Java"
         String subStr=str.substring(3, 11); // subStr --> "lenguaje"

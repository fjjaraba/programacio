8. Altres classes d'interès

Classe Math

https://docs.oracle.com/javase/8/docs/api/java/lang/Math.html

La classe Math resideix al paquet java.lang i proporciona un conjunt de mètodes estàtics per realitzar diversos càlculs matemàtics diferents. Com que és una classe estàtica no es poden instanciar objectes, així que per fer ús dels seus mètodes es posa el nom de la classe seguit d'un punt i el nom del mètode o atribut.

Math.metodo();

Atributs:

  • static double E  -->  representa el número irracional   e (2.718281828459045).
  • static double PI  -->  representa el valor pi (3.141592653589793).

Mètodes:

  • Trigonometría:
    • static double acos(double a)  -->  arcoscós d'un angle.
    • static double asin(double a)  -->  arcosé d'un angle.
    • static double atan(double a)  -->  arcotangent
    • static double atan2(double y, double x)  -->  Retorna l'angle theta de la conversió de coordenades rectangulars (x, y) a coordenades polars (r, theta).
    • static double cos(double a)  -->  cosinus d'un angle
    • static double cosh(double x)  -->  cosinus hiperbólic
    • static double sin(double a)  -->  devuelve el seno de un ángulo
    • static double sinh(double x)  -->  devuelve el seno hiperbòlico
    • static double tan(double a)  -->  tangent d'un angle..
    • static double tanh(double x)  -->  tangent hiperbòlica.
    • static double toDegrees(double angrad)  -->  converteix un angle expressat en radians a graus..
    • static double toRadians(double angdeg)  -->  converteix un angle de graus a radians..
  • Arredoniment, truncament:
    • static double ceil(double a)  -->  Arrodoneix cap amunt, és a dir, torna l'enter més petit que és major o igual al nombre donat.
    • static double floor(double a)  -->  Arredoneix cap avall, és a dir, torna l'enter més gran que és menor o igual al nombre donat.
    • static int floorDiv(int x, int y)  -->  Retorna el valor int més gran (més proper a l'infinit positiu) que sigui menor o igual al quocient algebraic.
      static long floorDiv(long x, long y)
    • static int floorMod(int x, int y)  -->  Retorna el mòdul dels arguments int ( x % y )
    • static long floorMod(long x, long y)
    • static double nextAfter(double start, double direction)  -->  Retorna el nombre adjacent a start en la direcció de direction.
    • static float nextAfter(float start, double direction)
    • static double nextDown(double d)  -->  retorna el nombre inferior..
    • static float nextDown(float f)
    • static double nextUp(double d)  -->  retorna el nombre superior..
    • static float nextUp(float f)
    • static double rint(double a)  -->  Retorna el valor double més proper en valor a l'argument i igual a un enter matemàtic..
    • static long round(double a)  -->  arrodoneix l'enter més proper.
    • static int round(float a)
  • Potències, arrels, logaritmes:
    • static double cbrt(double a)  --> arrel cúbica.
    • static double exp(double a)  -->   Retorna el nombre d'Euler, e, elevat a la potència d'un valor doble.
    • static double expm1(double x)  -->  torna ex-1
    • static int getExponent(double d)  -->  Retorna l'exponent utilitzat en la representació d'un doble
    • static int getExponent(float f)
    • static double log(double a)  -->  torna el logaritme en base e.
    • static double log10(double a)  --> torna el logaritme en base  10.
    • static double pow(double a, double b)  -->  torna ab.
    • static double sqrt(double a)  -->  torna l'arrel quadrada positiva.
  •  màxims, mínims, següents, anteriors, sumes, restes, multiplicacions, divisions:
    • static int addExact(int x, int y)  -->  suma de dos arguments.
    • static long addExact(long x, long y)
    • static int decrementExact(int a)  -->  Retorna l'argument disminuït en un, llançant una excepció si el resultat desborda un int.
    • static long decrementExact(long a)
    • static int incrementExact(int a)  -->  torna l'argument incrementat en 1 (produeix excepció si hi ha desbordament).
    • static long incrementExact(long a)
    • static double max(double a, double b)  -->  més gran de 2 valors.
    • static float max(float a, float b)
    • static int max(int a, int b)
    • static long max(long a, long b)
    • static double min( a,  b)  -->  menor de 2 valors
    • static int multiplyExact( x,  y)  -->  torna la multiplicació. Si soprepasseu el màxim valor produeix una excepció.
    • static int subtractExact(int x, int y)  -->  diferència x - y. Si es produeix overflow, produeix una excepció..
    • static long subtractExact(long x, long y)
  • Altres:
    • static double abs(double a)  -->  valor absolut
    • static float abs(float a)
    • static int abs(int a)
    • static long abs(long a)
    • static double copySign(double magnitude, double sign)  -->  Retorna el primer argument de punt flotant amb el signe del segon argument de punt flotant..
    • static float copySign(float magnitude, float sign)
    • static double random()  -->  retorna un nombre aleatori entre [0.0 i 1.0[
    • static double signum(double d)  -->  retorna el signe: 0 si d és zero; 1 si d és positiu; -1 si d és negatiu.
    • static float signum(float f)
    • static int toIntExact(long value)  -->  converteix int. Si hi ha overflows produeix una excepció.

Exemple:

public class App {
  public static void main(String[] args) {
    double valor = 2.3265874521365;
    int base = 5, exponent = 3;
    System.out.println("Número: " + valor);
    System.out.println("Arredoneix a 2 decimals cap avall: " + Math.floor(valor * 100)/100.0);
    System.out.println("Arrodoneix a 2 decimals cap amunt: " + Math.ceil(valor * 100)/100.0);
    System.out.println("Arrodoneix a 2 decimals: " + Math.round(valor * 100)/100.0);
    System.out.println(base + " elevat a " + exponente + " = " + Math.pow(base, exponent));
    System.out.println("Tirada de dau: " + (int)(1 + Math.round((Math.random() * (6 - 1)))));
  }
}

Classe Random

https://docs.oracle.com/javase/8/docs/api/java/util/Random.html

S'utilitza per generar un flux de nombres pseudoaleatoris. La classe utilitza una llavor de 48 bits per generar els números. Si es creen dues instàncies de Random amb la mateixa llavor i es realitza la mateixa seqüència de trucades de mètode per a cadascuna, generaran i tornaran seqüències idèntiques de números. Es troba a la llibreria java.util

Constructors:

  • Random() --> crea un objecte Random
  • Random(long seed) --> crea un objecte utilitzant una llavor.

Mètodes:

  • DoubleStream doubles() --> retorna un stream il·limitat de valors double pseudoaleatoris entre 0 (inclusivament) i 1 (exclòs), [0, 1[
    DoubleStream doubles(double origen, double limite) --> retorna un stream il·limitat de valors entre [origen, limite[
    DoubleStream doubles(long streamSize) --> retorna un stream de doubles amb un nombre concret de valors entre [0, 1[
    DoubleStream doubles(long mida, double origen, double limite)
  • IntStream ints() --> torna un stream il·limitat d'enters.
    IntStream ints(int origen, int limite) --> stream d'enters entre [origen, limite[
    IntStream ints(long n) --> stream d'enters amb n enters.
    IntStream ints(long n, int origen, int limite)
  • LongStream longs()
    LongStream longs(long streamSize)
    LongStream longs(long randomNumberOrigin, long randomNumberBound)
    LongStream longs(long streamSize, long randomNumberOrigin, long randomNumberBound)
  • protected int next(int bits) --> genera el següent nombre pseudoaleatori.
  • boolean nextBoolean() --> torna el següent boolean de forma pseudoaleatòria.
  • void nextBytes(byte[] bytes) --> Genera bytes aleatoris i els col·loca en una matriu de bytes proporcionada per l'usuari.
  • double nextDouble() --> retorna el següent double ( entre [0, 1[ ) de forma pseudoaleatòria.
  • float nextFloat()
  • int nextInt()
    int nextInt(int bound) --> següent sencer entre [0, bound[
  • long nextLong()
  • void setSeed(long seed) --> Estableix la llavor d'aquest generador de números aleatoris utilitzant una llavor passada com a paràmetre

Exemple:

import java.util.Random;
public class App {
  public static void main(String[] args) {
    Random randomNumbers = new Random();       // create random object
    System.out.println("---------- Boolean Aleatorio ---------" );
    /* Devuelve el siguiente boolean pseudoaleatorio que se usa en un sorteo  */
    boolean valor = randomNumbers.nextBoolean();
    System.out.println("Valor del booleano aleatorio: " + valor);
    /* Simula la tirada de un dato, dando valores entre 1 y 6. Dado que genera números entre [0 y 6[
    * al sumarle 1, el más pequeño será 1. Y puesto que 6 no sale nunca (el int más pequeño es 5), 5 + 1 dará 6 */
    System.out.println("\n---------- Tirada de 5 dados ---------" );
    System.out.println("Dado 1: " + (randomNumbers.nextInt(6)+1) );
    System.out.println("Dado 2: " + (randomNumbers.nextInt(6)+1) );
    System.out.println("Dado 3: " + (randomNumbers.nextInt(6)+1) );
    System.out.println("Dado 4: " + (randomNumbers.nextInt(6)+1) );
    System.out.println("Dado 5: " + (randomNumbers.nextInt(6)+1) );

    // devuelve el siguiente pseudoaleatorio long
    Long val = randomNumbers.nextLong();
    System.out.println("\n---------- Long aleatorio ---------" );
    System.out.println("valor Long aleatorio: " + val);      
  }
}

Classe DecimalFormat

https://docs.oracle.com/javase/8/docs/api/java/text/DecimalFormat.html

DecimalFormat (forma part del paquet java.text) és la classe java ens permet mostrar els números en pantalla amb el format que vulguem, per exemple, amb dos decimals, amb una coma per separar els decimals, etc. DecimalFormat també és útil per presentar un número en un JTextField o recollir el text del JTextField i reconstruir-ne el nombre.

Per donar format a un número, hem de definir un patró, que és una seqüència de caràcters especials que es poden barrejar amb text. Els caràcters especials són:

0   –-> escriu el dígit subministrat, 0 si no se'n subministra cap.
#   –-> escriu el dígit subministrat, res si no se subministra dígit.
.   –-> indica on posar el separador de decimals.
,   –-> indica on posar el separador de milers.
-   --> signe menys
E   --> Separa la mantisa i l'exponent en notació científica. No cal posar-los entre cometes en posició prefix o sufix.
;   --> separa patrons per a números positius i negatius.
%   --> ho multiplica per 100 i ho mostra com a percentatge.
\u2030   --> Multiplicar per 1000 i mostrar com a valor per mil
¤   --> símbol de moneda

Quan el patró s'aplica a un número, s'executen les regles de format i el resultat s'imprimeix d'acord amb el DecimalFormatSymbol de la configuració regional de la nostra JVM, tret que s'especifiqui una configuració regional específica.

Constructors:

  • DecimalFormat() --> crea un objecte usant el patró i símbols del format local.
  • DecimalFormat(String pattern) --> crea un objecte usant el patró passat i els símbols locals.
  • DecimalFormat(String pattern, DecimalFormatSymbols symbols) --> crea un objecte usant el patró i símbols passats.

Mètodes:

  • void applyLocalizedPattern(String pattern) --> aplica el patró.
  • void applyPattern(String pattern) --> aplica el patró.
  • StringBuffer format(double number, StringBuffer result, FieldPosition fieldPosition) --> formata un double
    StringBuffer format(long number, StringBuffer result, FieldPosition fieldPosition) --> formata un long
    StringBuffer format(Object number, StringBuffer toAppendTo, FieldPosition pos) --> formata un objecte numèric
  • Currency getCurrency() --> Obté la moneda utilitzada per aquest format decimal en formatar valors de moneda.
  • int getGroupingSize() --> mostra el número que s'utilitza per agrupar els dígits (en 1,123.658.235.23 estan agrupats cada 3 dígits).
  • int getMultiplier() --> torna el multiplicador usat en els percentatges, milers i formats similars.
  • String getNegativePrefix() --> obté el prefix per a valors negatius ( signe - )
  • String getPositivePrefix() --> obté el prefix per a valors positius
  • void setGroupingSize(int newValue) --> estableix el nombre de dígits que s'agruparan.
  • void setMultiplier(int newValue) --> estableix el multiplicador usat en els percentatges, milers i formats similars
  • void setNegativePrefix(String newValue) --> canvia el símbol per als números negatius.
  • void setPositivePrefix(String newValue) --> canvia el símbol per als números positius.

Arrodonir decimals

DecimalFormat formateador = new DecimalFormat("####.##");
System.out.println(formateador.format(3.43242383));      // 3,43 
System.out.println(formateador.format(58754.43522383));  // 5875,.44 
System.out.println(formateador.format(1284.43492383));   // 1284,43

Les  #  representen una xifra, així que hem posat que el número es posi en pantalla amb quatre xifres (si la part sencera té més de 4 igual es mostren), un punt decimal i dos decimals (s'arrodoneix només en funció del decimal inferior, arrodoniment matemàtic).

Si fem servir 0 en comptes de #, els buits s'emplenaran amb zeros per davant.

DecimalFormat formateador = new DecimalFormat("0000.00");
System.out.println(formateador.format(3.43242383));     // 3,43 
System.out.println(formateador.format(58754.43522383)); // 5875,44 
System.out.println(formateador.format(1284.43492383));  // 1284,43
System.out.println(formateador.format(1284.4));         // 1284,40

las cifras que falten por delante o por detrás, se rellenarán con ceros.

Mostrar percentatges

DecimalFormat formateador = new DecimalFormat("00.##%");
System.out.println (formateador.format (0.43242383)); // 43,24% 
System.out.println (formateador.format (0.43522383)); // 43,52% 
System.out.println (formateador.format (1.43492383)); // 143,49%
System.out.println (formateador.format (1.4));        // 140%;

Notació científica

DecimalFormat formateador = new DecimalFormat("00.#######E0"); 
System.out.println (formateador.format (0432423.83));     // 43,242383E4 
System.out.println (formateador.format (-04352238.3));    // -43,522383E5 
System.out.println (formateador.format (0.000143492383)); // 14,3492383E-5

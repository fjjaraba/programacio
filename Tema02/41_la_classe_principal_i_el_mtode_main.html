4.1. La classe principal i el métode main

Un programa pot construir-se emprant diverses classes. En el cas més simple s'utilitzarà una única classe. Aquesta classe conté el programa, rutina o mètode principal: main() i en aquest s'inclouen les sentències del programa principal.

public class ClassePrincipal {
   public static void main(String[] args) {
      sentencia_1;
      sentencia_2;
      // ...
      sentencia_N;
   }
}

Com s'ha indicat anteriorment, en un programa de Java tot s'organitza dins de les classes. En l'exemple anterior, ClassePrincipal és el nom de la classe principal i de l'arxiu que conté el codi font. Tots els programes o aplicacions independents escrites en Java tenen un mètode main o principal que, al seu torn, conté un conjunt de sentències. Aquest mètode és el punt d'arrancada de l'execució de tot programa en Java.

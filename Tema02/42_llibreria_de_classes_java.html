4.2. Llibreria de classes Java

Java proporciona més de 2000 classes predefinides que podeu utilitzar per afegir funcionalitat al vostre programa. Algunes de molt útils són:

  • String, que proporciona un tipus de dades per a cadenes de caràcters, juntament amb mètodes per buscar i manipular cadenes
  • Random, que genera números aleatoris
  • Scanner, que proporciona mètodes per llegir l'entrada des de la consola Java.
  • System y PrintStream, que proporcionen membres de dades i mètodes per imprimir dades a la consola Java.
  • DecimalFormat y NumberFormat, que permeten formatar números per a la sortida
  • Math, que proporciona mètodes per realitzar operacions matemàtiques
  • Envoltoris d'objetes (Integer, Byte, Short, Long, Float, Double, Boolean, Character...), que proporcionen un objecte equivalent als tipus de dades primitives perquè puguin utilitzar-se al programa com si fossin objecte.
  • JOptionPane, que us permet utilitzar quadres de diàleg per mostrar missatges a l'usuari o per obtenir informació de l'usuari.

Les classes Java estan organitzades en paquets, agrupats segons la seva funcionalitat.

  • java.lang  -->  Funcionalitat bàsica comú a molts programes, comolas classe String, Math o els envoltoris d'objectes.
  • java.awt  -->  Classes de gràfics per a dibuixar i usar colors i components de interficies d'usuari.
  • javax.swing  -->  Components d'interficie d'usuari de nou estil que tenen una aparença consistent en totes las plataformes.
  • java.text  -->  Classes per formatar la sortida numèrica.
  • java.util  -->  Las classe Scanner, Random y otras diversas.
  • java.io  -->  Classes per a llegir i escriure en arxius.

Podem vore tots els paquets i classes a l'API de Java: https://docs.oracle.com/javase/8/docs/api/

Totes les classes incluides a java.lang estan automàticament disponibles als nostres programes en Java. Per a utilitzar una classe que no hi estiga al el paquet java.lang, cal indicar-li al compilador en quin paquet resideix la classe. Per a això, ha d'incloure una declaració d'importació en el seu programa:

import java.text.DecimalFormat;  // importa la classe DecimalFormat
import java.text.*;              // importa totes les classes incloses a java.text

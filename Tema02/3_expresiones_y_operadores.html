3. Expresiones y operadores

Expresiones

Una expresión consta de operadores y operandos ( valores, variables, operadores e invocaciones de métodos) que se evalúan como un único valor. Las expresiones se pueden utilizar en varios contextos, como en asignaciones, declaraciones condicionales y bucles.

Operadores

Los operadores son elementos que permiten realizar diversas operaciones y manipulaciones en datos y variables. Estas operaciones pueden abarcar desde simples cálculos matemáticos hasta evaluaciones de condiciones lógicas y manipulación de datos a nivel de bits. Son símbolos o palabras clave que se utilizan para llevar a cabo estas acciones de manera eficiente. Veamos algunos de los operadores más importantes:

  • Operadores Aritméticos

Son símbolos especiales que se utilizan para realizar operaciones matemáticas en variables numéricas. Siempre requieren dos operandos por lo que todos son operadores binarios.

Operador Nombre Función Ejemplo
+ Suma Devuelve el resultado de sumar los dos operandos. 5 + 3
- Resta Devuelve el resultado de restar al operando de la izquierda el operando de la derecha. 15 - 12
* Multiplicación Devuelve el resultado de multiplicar los dos operandos. 3 * 9
/ División Devuelve el cociente resultante de dividir el operando de la izquierda entre el operando de la derecha. Si los dos operandos son enteros, el resultado también será entero. 12 / 5
% Módulo

Devuelve el resto de la división entre enteros al dividir el operando izquierdo entre el derecho. Si el operador módulo se utiiza con operando reales devolverá también devuelve el resto. El cociente de la división, al utilizar el operador %, siempre es un número entero.

 12 % 5

Ejemplo:

/* Operadores aritméticos  */
public class ArithmeticOperators  {
  public static void main( String [] args )   {    
    double salad = 5.95;              // calculate the cost of lunch
    double water = .89;
    System.out.println( “The cost of lunch is $”  + ( salad + water ) );  
    int targetYear = 2011;          // calculate your age as of a certain year
    int birthYear = 1993;
    System.out.println( “Your age in “ + targetYear + “ is “  + ( targetYear - birthYear ) );

    int caloriesPerApple = 127;     // calculate the total calories of apples
    int numberOfApples = 3;
    System.out.println( “The calories in “ + numberOfApples  + “ apples is “ +  ( caloriesPerApple * numberOfApples ) );
   
    double miles = 426.8;         // calculate miles per gallon
    double gallons = 15.2;
    double mileage = miles / gallons;
    System.out.println( “The mileage is “  + mileage + “ miles per gallon.” );
  }
}

  • Operadores de asignación

Son fundamentales en Java ya que permiten asignar valores a variables. A través de estos operadores, se puede almacenar información en variables para su posterior uso en el programa. El operador más comúnmente utilizado es el signo igual ( = ), que asigna el valor de la derecha a la variable de la izquierda. Por ejemplo,      int x = 10;       asigna el valor 10 a la variable x.

A la izquierda del operador de asignación siempre debe de haber una variable o constante.  También se permiten asignaciones mútiples como las siguientes:

    c = a = b;          //equivalente a        c=(a=b);
    c = a = b = 321;    //asigna 321 a a, b y c

El operador asignación se puede combinar con los operadores aritméticos, permitiendo modificar el valor de la variable al mismo tiempo que le asigna dicho nuevo valor.

Operador asignación compuesto Equivalente
+= v += 3; v = v + 3;
-= v -= 3; v = v - 3
*= v *= 5; v = v * 5;
/= v /= 2; v = v / 2;
%= v %= 5; v = v % 5;

  • Operadores de comparación

Son esenciales para evaluar condiciones en programas Java. Comparan dos valores y devuelven un resultado booleano, es decir, true si la comparación es verdadera y false si es falsa.

Operador Nombre Ejemplo
== Igualdad a == b
!= Distintos a != b
> Mayor que a > b
< Menor que a < b
>= Mayor o igual que a >= 20 * 2
<= Menor o igual que edad <= 17

  • Operadores lógicos

Permiten combinar o invertir condiciones lógicas en un programa Java. Los operadores lógicos más comunes son && (AND lógico), || (OR lógico) y ! (NOT lógico).

En la tabla anterior A y B son dos expresiones que pueden ser evaluada como true o false.  Podemos observar que:

  • El operador && devuelve true si ambas condiciones son verdaderas.
  • El operador || devuelve true si al menos una condición es verdadera.
  • El operador  !   invierte el valor de una condición, es decir, cambia true a false y viceversa.

  • Operadores de incremento y decremento

Los operadores de incremento ( ++ ) y decremento ( --) son útiles para modificar el valor de una variable en una unidad. El operador de incremento aumenta el valor de la variable en 1, mientras que el operador de decremento lo disminuye en 1. Estos operadores son unarios pues trabajan con un solo operando que debe ser una variable, nunca un literal o una constante.

Es muy importante el lugar en el que se coloca el operador:

- Delante del operando

Primero modifica el valor de la variable y luego devuelve el valor resultante para que pueda ser utilizada en el cálculo de la expresión. Ejemplo:

a = 15;
b = ++a;        // a -> 16;   b -> 16
c = 2 * ++a;    // a -> 17;   c -> 34

Al finalizar la ejecución de las tres líneas, a tendrá el valor 17;   b tendrá el valor 16 y c tendrá el valor 34.

- Detrás del operando

Primero se devuelve el valor de la variable para que pueda ser utilizado en el cálculo de la expresión y finalmente se modifica el valor de la variable. Ejemplo:

a = 15;
b = a--;
c = 2 * a--;

Al finalizar la ejecución de las tres líneas, a tendrá el valor 13;   b tendrá el valor 15 y c tendrá el valor 28 ( 2 * 14).

  • Operadores de concatenación

En el contexto de cadenas de texto (String), Java proporciona el operador + para la concatenación de cadenas. Este operador se utiliza para unir dos o más cadenas de texto en una sola cadena.

String nombre = "Juan";
String apellido = "Pérez";
String nombreCompleto = nombre + " " + apellido;      // nombreCompleto -> "Juan Pérez"

También se puede utilizar el método concat() de la clase String para concatenar cadenas.

  • Operadores de bits

Se utilizan para realizar operaciones a nivel de bits en valores enteros. Estos operadores permiten manipular datos binarios y realizar operaciones de desplazamiento de bits. Ejemplos de operadores de bits incluyen  &  (AND a nivel de bits),  |  (OR a nivel de bits),  ^  (XOR a nivel de bits),  <<  (desplazamiento a la izquierda) y  >>  (desplazamiento a la derecha).

Operador Nombre Función Ejemplo
&

AND bit a bit
(binario)

 Realiza una operación AND bit-a-bit entre dos números. Para cada posición del valor binario realiza un AND con los dos bits de la misma posición de los operandos. 
   10  ->  0 0 0 0 1 0 1 0
   67  ->  0 1 0 0 0 0 1 1
10 & 67 -> 0 0 0 0 0 0 1 0 -> 2
| OR bit a bit
(binario)		 Realiza una operación OR bit-a-bit entre dos números. Para cada posición del valor binario realiza un OR con los dos bits de la misma posición de los operandos. 
   10  ->  0 0 0 0 1 0 1 0
   67  ->  0 1 0 0 0 0 1 1
10 | 67 -> 0 1 0 0 1 0 1 1 -> 75
^ XOR bit a bit
(binario)		 Realiza una operación XOR bit-a-bit entre dos números. El resultado es true solo cuando los dos operandos son diferentes (uno true otro false). Para cada posición del valor binario realiza un XOR con los dos bits de la misma posición de los operandos. 
   10  ->  0 0 0 0 1 0 1 0
   67  ->  0 1 0 0 0 0 1 1
10 ^ 67 -> 0 1 0 0 1 0 0 1 -> 73
~ Complemento a uno
(unario)		 Invierte todos los bits de un número. Afecta al signo (complemento a 1). 
  10 -> 0 0 0 0 1 0 1 0
 ~10 -> 1 1 1 1 0 1 0 1 -> -11 
<< Desplazamiento izquierda
(binario)		 Desplaza hacia la izquierda los bits del operando de la izquierda tantos lugares como indique el operando de la derecha. El relleno por la derecha se hace con ceros.
     valor_desplazar << num_posiciones		 
     10 -> 0 0 0 0 1 0 1 0
10 << 2 -> 0 0 1 0 1 0 0 0 -> 40 
>>

Desplazamiento derecha
(binario)

 Desplaza hacia la derecha los bits del operando de la izquierda tantos lugares como indique el operando de la derecha.
     valor_desplazar >> num_posiciones		 
     10 -> 0 0 0 0 1 0 1 0
10 >> 2 -> 0 0 0 0 0 0 1 0 -> 2 
>>> Desplazamiento derecha sin signo Similar al operador >> (desplazamiento a la derecha), pero siempre agrega un 0 a la izquierda en lugar del bit de signo. Esto significa que, independientemente del signo del número, se realiza el desplazamiento de bits a la derecha. 
 int a = -10;
 int b = a >>> 2;   // b -> 1073741821

Precedencia y Asociatividad

Cuando en una expresión hay varios operadores es importante saber cuál se ejecuta primero. Veamos algún ejemplo:

2 * 3 + 5

  • Si primero se hace la suma el resultado es 16 (2 * 8).
  • Si primero se hace la multiplicación, el resultado es 11 (6 + 11).

La precedencia es la prioridad a ejecutarse. Cada operador tiene un valor de precedencia. Cuando en una expresión hay operadores distintos, se ejecuta primero el de mayor precedencia. En el caso anterior, la multiplicación tiene un nivel de precedencia mayor que la suma, por lo que el resultado será 11. Veamos otro ejemplo.

32 / 8 / 2

  • Si primero se ejecuta la división 8 / 2 y a continuación 32 / 4, el resultado será 8.
  • Si primero se ejecuta la división 32 / 8 y a continuación 4 / 2, el resultado será 2.

La asociatividad es la dirección (de izq ->der o de der -> izq) en la que se ejecutan los operadores del mismo nivel en una expresión. Lo más frecuente es de izquierda a derecha. En el caso anterior el resultado será 2.

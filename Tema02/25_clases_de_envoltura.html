2.5. Clases de envoltura

Class Integer

https://docs.oracle.com/javase/8/docs/api/java/lang/Integer.html

La clase Integer envuelve un valor del tipo primitivo int en un objeto. Un objeto de tipo Integer contiene un único campo cuyo tipo es int. Además, esta clase proporciona varios métodos para convertir un int en una cadena y una cadena en un int, así como otras constantes y métodos útiles cuando se trabaja con un int.

Atributos:

  • static int BYTES   -->  Número de bytes utilizados para almacenar el número en binario.
  • static int MAX_VALUE  -->  valor máximo que puede tomar:   231-1.
  • static int MIN_VALUE  -->  valor mínimo que puede tomar:    -231.
  • static int SIZE  -->  número de bits usados para almacenar el número en binario.
  • static Class<Integer> TYPE  -->  La instancia de clase que representa el tipo primitivo int

Instanciación:

Boolean objeto = new Integer( valorint);
Boolean objeto = new Integer("345");

Constructor:

  • Integer(int value)  -->  crea un objeto Integer a partir de un tipo primitivo int.
  • Integer(String s)  -->  crea un objeto Integer a partir de un String que contiene caracteres numéricos.

Métodos:

  • static int bitCount(int i)  -->  devuelve el número de bits a 1 en binario de i.
  • byte byteValue()  -->  convierte a byte.
  • static int compare(int x, int y) -->  compara 2 valores int. Devuelve 0 si x es igual a y; -1 si x < y;  1 si x > y.
  • int compareTo(Integer anotherInteger)  -->  compara el objeto actual con el argumento pasado.
  • static int compareUnsigned(int x, int y)  -->  compara tratando los valores como enteros sin signo.
  • static Integer decode(String nm)  -->  convierte un String con contenido un entero a Integer.
  • static int divideUnsigned(int dividend, int divisor)  -->  divide dos enteros sin signo.
  • double doubleValue()  -->  convierte a  double.
  • boolean equals(Object obj)  -->  compara el objeto actual con el argumento.
  • float floatValue()  -->  convierte a float.
  • int hashCode()  -->  devuelve el código hash del Integer.
  • static int hashCode(int value)  -->  devuelve el código hash de value.
  • int intValue()  -->  convierte a int
  • long longValue()  -->  convierte a long.
  • static int max(int a, int b)  -->  devuelve el mayor de 2 int.
  • static int min(int a, int b)  -->  devuelve el menor de 2 int.
  • static int parseInt(String s)  -->  convierte el argumento s en un int.
  • static int remainderUnsigned(int dividend, int divisor)  -->  devuelve un entero sin signo resultante de la división de los argumentos.
  • static int reverse(int i)  -->  Devuelve el valor obtenido al invertir el orden de los bits en la representación binaria de complemento a dos del valor int especificado.
  • static int reverseBytes(int i)  -->  Devuelve el valor obtenido al invertir el orden de los bytes en la representación del complemento a dos del valor int especificado.
  • static int rotateLeft(int i, int distance)  -->  Devuelve el valor obtenido al rotar la representación binaria de complemento a dos del argumento "i" a la izquierda por el número especificado de bits indicado en distance.
  • static int rotateRight(int i, int distance)  -->  rota a la derecha los bits indicados en distance.
  • short shortValue()  -->  convierte a short.
  • static int sum(int a, int b)  -->  devuelve la suma de dos int.
  • static String toBinaryString(int i)  -->  devuelve una cadena con la representación del entero en complemento a 2.
  • static String toHexString(int i)  -->  devuelve una cadena con la representación de "i" en hexadecimal.
  • static String toOctalString(int i)  -->  devuelve una cadena con la representación de "i" en octal.
  • String toString()  -->  convierte a String.
  • static String toString(int i)  -->  convierte "i" a String.
  • static String toString(int i, int radix)  -->  devuelve un String resultado de convertir "i" a la base indicada por radix.
  • static long toUnsignedLong(int x)   -->  convierte x a long sin signo.
  • static String toUnsignedString(int i)  -->  devuelve una cadena que representa un valor sin signo.
  • static Integer valueOf(int i)  -->   convierte int a Integer.
  • static Integer valueOf(String s)  -->  convierte String a Integer. El contenido de s deben ser caracteres numéricos.
  • static Integer valueOf(String s, int radix)  -->  convierte el String  en base radix a Integer.

Byte, Short, Long

https://docs.oracle.com/javase/8/docs/api/java/lang/Byte.html

https://docs.oracle.com/javase/8/docs/api/java/lang/Short.html

https://docs.oracle.com/javase/8/docs/api/java/lang/Long.html

La clase Byte envuelve un valor de tipo primitivo byte en un objeto. Un objeto de tipo Byte contiene un único campo cuyo tipo es byte.

La clase Short envuelve un valor de tipo primitivo short en un objeto. Un objeto de tipo Short contiene un único campo cuyo tipo es short.

La clase Long envuelve un valor de tipo primitivo long en un objeto. Un objeto de tipo Short contiene un único campo cuyo tipo es short.

Atributos:

  • static int BYTES  -->  números de bytes utilizados para la representación en binario.
  • static byte MAX_VALUE  --> valor máximo
  • static byte MIN_VALUE  -->  mínimo
  • static int SIZE  -->  número de bits.
  • static Class<Byte> TYPE  -->  instancia de la clase que representa al tipo primitivo byte, short, long.

Constructor

Byte Short Long
  • Byte(byte value) 
  • Byte(String s)
  • Short(short value) 
  • Short(String s)
  • Long(long value) 
  • Long(String s)

Métodos:

Muy parecidos a Integer

Double / Float

https://docs.oracle.com/javase/8/docs/api/java/lang/Double.html

https://docs.oracle.com/javase/8/docs/api/java/lang/Float.html

La clase Double envuelve un valor del tipo primitivo double en un objeto. Un objeto de tipo Double contiene un único campo cuyo tipo es double.

La clase Float envuelve un valor de tipo primitivo float en un objeto. Un objeto de tipo Float contiene un único campo cuyo tipo es float.

Atributos:

  • static int BYTES  -->  número de bytes utilizados pra representar el valor double.
  • static int MAX_EXPONENT  -->  Exponente máximo
  • static double/float MAX_VALUE  -->  valor positivo máximo.
  • static int MIN_EXPONENT  -->  exponente mínimo.
  • static double/float MIN_NORMAL  -->  valor positivo más pequeño.
  • static double/float MIN_VALUE  -->  valor positivo más pequeño distinto de cero.
  • static double/float NaN  -->  constante que almacena el valor NaN (Not-a-Nuber) de tipo double/float.
  • static double/float NEGATIVE_INFINITY  -->  constante que almacena el infinito negativo.
  • static double/float POSITIVE_INFINITY  -->  infinito positivo.
  • static int SIZE  -->  número de bits utilizado para representar el valor.

Constructores:

Double Float
  • Double(double value)
  • Double(String s)
  • Float(double value)
  • Float(float value)
  • Float(String s)

Métodos:

Muy parecidos a Integer. Además podemos encontrar:

  • static boolean isFinite(double d)  -->  Devuelve true si el argumento es un valor de punto flotante finito; devuelve false en caso contrario (para argumentos NaN e infinito).
  • boolean isInfinite()  -->  true si el objeto es una magnitud infinita, false en caso contrario.
  • static boolean isInfinite(double v)  -->  true si v es infinita, false en caso contrario.
  • boolean isNaN()  -->  true si el objeto es un valor NaN.
  • static boolean isNaN(double v)  -->  true si v es un valor NaN.

Boolean

https://docs.oracle.com/javase/8/docs/api/java/lang/Boolean.html

La clase Boolean envuelve un valor del tipo primitivo boolean en un objeto. Un objeto de tipo Boolean contiene un único campo cuyo tipo es boolean.

Atributos:

  • static Boolean FALSE  -->  equivale a false.
  • static Boolean TRUE  -->  equivale a true.

Constructores:

  • Boolean(boolean value)
  • Boolean(String s)

Instanciación:

  • Boolean objeto = new constructor;

Métodos:

Algunos métodos son similares a Integer. Además tiene:

  • boolean booleanValue()  -->  convierte a boolean.
  • static int compare(boolean x, boolean y)  -->  compara 2 boolean y devuelve 0, -1, o 1.
  • int compareTo(Boolean b)  -->  compara el objeto actual con el objeto pasado como parámetro.
  • boolean equals(Object obj)
  • static boolean logicalAnd(boolean a, boolean b)  -->  devuelve el resultado de aplicar a los parámetros el operador AND.
  • static boolean logicalOr(boolean a, boolean b)  --> devuelve el resultado de aplicar a los parámetros el operador OR.
  • static boolean logicalXor(boolean a, boolean b)  -->  devuelve el resultado de aplicar a los parámetros el operador XOR.

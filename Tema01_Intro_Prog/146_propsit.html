1.4.6. Propòsit

També podem classificar els llenguatges segons l'ús que se'ls done. Així trombem:

Llenguatges d'ús general

Són llenguatges que han sigut pensats per a resoldre múltiples problemes. Java, C i Python són llenguatges de propòsit general.

Llenguatges d'ús específic

Són llenguatges que estan pensats per a una cosa en concret i que no té sentit usar fora del seu àmbit d'actuació. Per exemple, SQL és un llenguatge específic.

Java

Java és un llenguatge de propòsit general. Els principals utilitats que se'ls dona als programes de Java són:

  • Videojocs. Aquest llenguatge s'usa per a desenvolupar videojocs, jocs per a cel·lular i computadors, inclosos aquells que integren tecnologies com machine learning i altres relacionades amb la realitat virtual.
  • Webs. Java ofereix tot el necessari per al desenvolupament web, però també existeix una gran varietat de frameworks.
  • Aplicacions mòbils. Java és el llenguatge amb el qual es programen les aplicacions natives per a Android, però també aquelles per a IOS i altres sistemes operatius.
  • Aplicacions per a Smart TV. Moltes aplicacions que empren les televisions intel·ligents són desenvolupades amb aquest llenguatge.
  • Computació en el núvol. A l'ésser un llenguatge multiplataforma, Java és molt útil per a fer aplicacions descentralitzades basades en el núvol.
  • Big Data. Serveix per a fer els motors de processament de dades, que analitzen quantitats massives d'informació en temps real.
  • Intel·ligència artificial (IA). Aquest llenguatge també s'utilitza en el machine learning, el llenguatge natural i l'aprenentatge profund, entre altres avanços vinculats amb la IA.
  • Internet of things (IoT). Aquest és un altre dels grans avanços tecnològics de l'actualitat en els quals s'usa Java. Entre altres coses, es destina a la programació dels sensors en dispositius de perifèria, que poden connectar-se a la xarxa de manera independent.

1. Conceptes bàsics d'informàtica

Hardware

Com es mostra en la Figura, un ordinador normalment inclou els següents components:

  • Una CPU, o unitat central de processament, que executa les instruccions d'un programa.
  • Una memòria, que conté les instruccions i les dades d'un programa mentre s'executa.
  • Un disc dur, utilitzat per emmagatzemar programes i dades perquè puguen ser carregats en la memòria i accedits per la CPU.
  • Un teclat i un ratolí, utilitzats per a l'entrada de dades.
  • Un monitor, utilitzat per mostrar la eixida d'un programa.
  • Un port Ethernet per connectar-se a Internet o a una xarxa d'àrea local (LAN).
  • Altres components (no es mostren) com una targeta gràfica i un DVD

Les CPU consten d'una unitat aritmètica lògica (UAL) [també anomenada unitat entera (UE)], que realitza operacions lògiques i aritmètiques enteres bàsiques; una unitat de coma flotant (UCF), que realitza operacions aritmètiques de coma flotant; un conjunt de registres de maquinari per emmagatzemar dades i adreces de memòria; i un altre maquinari de suport, inclosa una unitat de
control per seqüenciar les instruccions. Cadascuna CPU ve amb el seu propi conjunt d'instruccions, que són les operacions que pot realitzar.

Sistemes Operatius

Un sistema operatiu (SO) és un programa de programari que:

  • controla els dispositius perifèrics (per exemple, administra el sistema de fitxers),
  • admet la multitasca, programant l'execució de diversos programes durant el mateix interval de temps,
  • assigna memòria a cada programa, de manera que no hi haja conflictes entre la memòria de qualsevol programa que s'estiga executant al mateix temps,
  • evita que l'usuari danye el sistema. Per exemple, evita que els programes de l'usuari sobreescriguen el sistema operatiu o
    la memòria d'un altre programa.

Windows ha evolucionat d'un sistema operatiu DOS monousuari i monotasca a un sistema multiusuari i multitasca.

Unix i Linux, d'altra banda, van ser dissenyats des del principi per a ser sistemes operatius multiusuari i multitasca.
